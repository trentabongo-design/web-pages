DNS - Domain Name System

What is DNS?

DNS (Domain Name System) is like a phone book for the internet. Every website has an IP address (a number like 142.250.74.46), but instead of remembering numbers, DNS lets you type a name like www.google.com and automatically translates it into the correct IP address so your browser can find the right server.

Structure

  • Domain name — the human-friendly name e.g. www.google.com
  • IP address — the numerical address computers actually use
  • DNS server — the system that translates one into the other
