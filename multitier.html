Multitier Architecture

What is Multitier Architecture?

Multitier architecture is a way of designing a software system by dividing it into separate layers, each with its own specific role. The most common version is the three-tier architecture.

The Three Layers

  • Presentation Layer — what the user sees on screen (the web page)
  • Logic Layer — the behind-the-scenes processing and rules
  • Data Layer — the database where information is stored
