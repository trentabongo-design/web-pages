Web Servers

What is a Web Server?

A web server is a computer that stores website files and sends them to users when requested. When you type a website address in your browser, the browser connects to that website's web server, which then sends back the page for your browser to display.

Examples of Web Server Software

  • Apache
  • Nginx
  • Microsoft IIS
