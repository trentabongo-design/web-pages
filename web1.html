Web 1.0 - The Static Web

What is Web 1.0?

Web 1.0 is the first generation of the World Wide Web, from around 1991 to 2004. It is called the "read-only web" because users could only read information on websites — they could not post, comment, or interact in any way. Pages were static, meaning they looked the same to every visitor and were only updated when a developer manually changed the files.
