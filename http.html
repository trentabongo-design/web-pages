HTTP - Hypertext Transfer Protocol

What is HTTP?

HTTP (Hypertext Transfer Protocol) is the set of rules that controls how web pages are transferred between a browser and a web server. Every time you visit a website, HTTP is what makes that communication happen. HTTPS is the secure version of HTTP, where all data is encrypted.

Structure

  • Request — the browser asks the server for a page
  • Response — the server sends the page back
  • HTTP — unsecured connection
  • HTTPS — secured and encrypted connection
