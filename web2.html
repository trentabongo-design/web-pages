Web 2.0 - The Social Web

What is Web 2.0?

Web 2.0 is the second generation of the internet, starting around 2004. Unlike Web 1.0 where users only read content, Web 2.0 allows users to create and share their own content. It is called the "read-write web." Examples include social media platforms like Facebook, YouTube, Twitter, and Wikipedia — all built around content created by users.
