Web 2.0 — The Social Web

Web Evolution — Term 14

What Is Web 2.0?

Web 2.0 refers to the second generation of the World Wide Web, emerging around 2004 and continuing largely to the present day. It is defined by a fundamental shift from passive consumption to active participation. Where Web 1.0 was read-only, Web 2.0 is read-write — users can now create, share, comment, collaborate, and interact with content and with each other.

The term "Web 2.0" was popularised by Tim O'Reilly and Dale Dougherty around 2004. It describes not just new technologies, but a new philosophy of how the web is used — one built around user-generated content, social interaction, and platforms that become more valuable as more people use them.

Characteristics of Web 2.0

Web 2.0 is characterised by platforms that enable user participation. Social media sites like Facebook, Twitter, and Instagram allow users to post content, follow others, and engage in conversations. YouTube allows anyone to publish videos. Wikipedia is written by its own users. Blogs allow individuals to publish their own writing and receive comments from readers.

Technically, Web 2.0 was enabled by the wider adoption of JavaScript (especially AJAX, which allows pages to update without fully reloading), improved CSS for richer layouts, and the proliferation of web APIs that allowed different services to integrate with each other.

The Impact of Web 2.0

Web 2.0 fundamentally changed how people use the internet. It gave rise to entirely new industries — social media, on-demand video, user review platforms, crowdfunding, and the sharing economy all emerged from Web 2.0's participatory model. The line between content producer and content consumer blurred almost entirely.

However, Web 2.0 also brought challenges. Centralisation of power in a few large platforms, data privacy concerns, misinformation, and the commodification of personal data all became significant issues, which fuelled discussions about what Web 3.0 should look like.

Key Facts About Web 2.0

  • Web 2.0 emerged around 2004 and is often called the "social" or "participatory" web.
  • It shifted the web from read-only to read-write — users can now create content.
  • Key platforms include Facebook, YouTube, Wikipedia, Twitter, and blogs.
  • Technologies like AJAX, REST APIs, and CSS3 powered the Web 2.0 experience.
  • User-generated content, collaboration, and community became central to the web.
  • Concerns about data privacy and platform monopolies also grew significantly.
