File Transfer Protocol (FTP)

File Transfer — Term 11

What Is FTP?

FTP stands for File Transfer Protocol. It is a standard network protocol used to transfer files from one computer to another over a TCP/IP network, such as the internet. It was one of the earliest internet protocols and was designed specifically to make file sharing between computers straightforward and reliable.

In simple terms, FTP allows you to upload files to a server or download files from a server. Web developers commonly used FTP to upload their website files to a web host. Even today, FTP (and its more secure successors) remains widely used in professional environments for large-scale file transfers.

How FTP Works

FTP uses a client-server model. The FTP client is the software on your computer that initiates the connection — programs like FileZilla are popular FTP clients. The FTP server is the remote computer that holds the files. When you connect, you authenticate with a username and password, and then you can browse the server's file system, upload files to it, or download files from it.

FTP uses two separate channels: a command channel (port 21) that handles instructions and responses, and a data channel (port 20) that handles the actual file transfer. This two-channel approach was innovative at the time it was designed.

FTP and Security

A significant drawback of standard FTP is that it transmits data — including usernames and passwords — in plain text, meaning anyone monitoring the network could intercept the information. For this reason, more secure alternatives have largely replaced it. SFTP (SSH File Transfer Protocol) and FTPS (FTP Secure) both encrypt the connection, making them far safer for transferring sensitive data.

Key Facts About FTP

  • FTP stands for File Transfer Protocol.
  • It is used to transfer files between computers over a network.
  • FTP uses port 21 for commands and port 20 for data transfer.
  • It was one of the first internet protocols, developed in the early 1970s.
  • Standard FTP is not encrypted and is considered insecure for sensitive transfers.
  • SFTP and FTPS are secure alternatives that encrypt the connection.
  • FileZilla is a popular free FTP client used by web developers worldwide.
