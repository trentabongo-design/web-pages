FTP - File Transfer Protocol

What is FTP?

FTP (File Transfer Protocol) is a set of rules used to transfer files between computers over a network. It allows you to upload files from your computer to a server, or download files from a server to your computer. Web developers commonly use FTP to upload website files to a hosting server.

Structure

  • FTP Client — software on your computer used to connect (e.g. FileZilla)
  • FTP Server — the remote computer holding the files
  • Upload — sending files from your computer to the server
  • Download — getting files from the server to your computer
