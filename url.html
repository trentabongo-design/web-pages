URL - Uniform Resource Locator

What is a URL?

A URL (Uniform Resource Locator) is the full address of a page or file on the internet. For example: https://www.example.com/about. Every web page, image, and file online has its own unique URL.

Structure of a URL

  • https:// — the protocol (how to connect)
  • www.example.com — the domain name (the website)
  • /about — the path (the specific page)
