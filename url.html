Uniform Resource Locator (URL)

Addressing — Term 5

What Is a URL?

A URL, which stands for Uniform Resource Locator, is the full address used to locate a specific resource on the internet. Every web page, image, video, or file on the internet has its own unique URL. When you type something like https://www.example.com/about into your browser, that entire string is a URL.

Think of a URL as a postal address for a resource on the internet. Just like a physical address tells the postal service exactly where to deliver a letter, a URL tells your browser exactly where to find a specific piece of content.

The Structure of a URL

A URL is made up of several parts. The first part is the scheme or protocol, such as https://, which tells the browser how to communicate with the server. Next comes the domain name, like www.example.com, which identifies the website. After the domain comes the path, such as /about, which points to a specific page or file on that server.

Some URLs also include a query string (starting with ?) that passes additional information to the server, and a fragment (starting with #) that points to a specific section within a page.

Why URLs Matter

URLs are the addressing system that makes the entire World Wide Web navigable. Without URLs, there would be no consistent way to link one page to another, share links, or tell a browser exactly what to load. Every hyperlink you click on a website is simply pointing to another URL.

Good URL design is also important for users and search engines. Clean, readable URLs like /products/shoes are far easier to understand than complex ones with random numbers and symbols.

Key Facts About URLs

  • URL stands for Uniform Resource Locator.
  • Every resource on the internet — pages, images, videos — has a unique URL.
  • A URL has several parts: scheme, domain, path, query string, and fragment.
  • URLs were defined by Tim Berners-Lee and the IETF in the early 1990s.
  • HTTPS URLs are encrypted, while HTTP URLs are not.
  • A URL is a specific type of URI (Uniform Resource Identifier).
