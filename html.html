HTML - Hypertext Markup Language

What is HTML?

HTML (Hypertext Markup Language) is the language used to create web pages. It uses tags to describe and structure content — things like headings, paragraphs, images, and links. Every website you visit is built using HTML as its base.

Basic Structure

  • <!DOCTYPE html> — tells the browser this is an HTML document
  • <html> — the root element wrapping the whole page
  • <head> — contains page information like the title
  • <body> — contains everything visible on screen
