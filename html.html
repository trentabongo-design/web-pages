Hypertext Markup Language (HTML)

Web Languages — Term 12

What Is HTML?

HTML stands for Hypertext Markup Language. It is the standard language used to create and structure content on the World Wide Web. Every web page you visit — including this one — is built using HTML as its foundation. It tells the browser what to display and how to structure it: what is a heading, what is a paragraph, where is an image, and what text should be a clickable link.

HTML is not a programming language in the traditional sense. It is a markup language, meaning it uses special tags to annotate and describe content rather than to perform calculations or control logic. Think of it as labelling the content of a document so a browser knows what each part means.

How HTML Works

HTML documents are made up of elements, which are represented by tags. Tags are written inside angle brackets, like <p> for a paragraph or <h1> for a top-level heading. Most elements have an opening tag and a closing tag, with the content placed in between — for example, <p>This is a paragraph.</p>.

Every HTML page has a basic structure: a <!DOCTYPE html> declaration at the top, an <html> element wrapping everything, a <head> section containing metadata (like the page title and linked stylesheets), and a <body> section containing the visible content.

HTML and Other Web Technologies

HTML rarely works alone on a modern website. It is almost always used alongside CSS (Cascading Style Sheets), which handles the visual design and layout, and JavaScript, which handles interactivity and dynamic behaviour. HTML provides the structure, CSS provides the style, and JavaScript provides the behaviour. Together, these three technologies power nearly every website in existence.

The current version of HTML is HTML5, which introduced many powerful new features including support for video and audio embedding, new semantic elements, and improved support for mobile devices.

Key Facts About HTML

  • HTML stands for Hypertext Markup Language.
  • It is the standard language for creating and structuring web pages.
  • HTML uses tags (like <h1>, <p>, <img>) to describe content.
  • It was created by Tim Berners-Lee in 1991 alongside the World Wide Web.
  • The current version, HTML5, was officially published in 2014.
  • HTML works alongside CSS for styling and JavaScript for interactivity.
  • Browsers interpret HTML and render it as the visual pages users see.
