World Wide Web (WWW)

Internet & Networking — Term 1

What Is the World Wide Web?

The World Wide Web, commonly referred to as the Web or WWW, is an information system that allows documents and other resources to be accessed over the internet. These resources are identified by URLs (web addresses), linked together by hyperlinks, and can be accessed using a web browser.

It is important to understand that the World Wide Web is not the same thing as the internet. The internet is the global network of connected computers, while the Web is just one of the services that runs on top of that network. Think of the internet as the roads, and the Web as one of the vehicles that travels on those roads.

A Brief History

The World Wide Web was invented by Sir Tim Berners-Lee in 1989 while he was working at CERN, the European nuclear research organization in Switzerland. He proposed a system for sharing information between scientists using hypertext — text that contains links to other texts.

By 1991, the first website had gone live. By the mid-1990s, the Web had grown explosively and became the dominant way people accessed information, communicated, and conducted business online. Today it hosts billions of pages covering virtually every topic imaginable.

How Does It Work?

When you type a web address into your browser, several things happen behind the scenes. Your browser sends a request to a server — a computer that stores the website — asking for a specific page. The server responds by sending back the files (HTML, images, etc.) that make up that page, and your browser displays them for you.

This back-and-forth communication is governed by a protocol called HTTP (Hypertext Transfer Protocol), which defines the rules for how requests and responses are formatted and sent.

Key Facts About the WWW

  • Invented by Tim Berners-Lee in 1989 at CERN.
  • The first website went live on August 6, 1991.
  • The Web is a service that runs on the internet — they are not the same thing.
  • Web pages are written in HTML and linked together using hyperlinks.
  • Communication between browsers and servers uses the HTTP protocol.
  • Today there are over 1.9 billion websites on the World Wide Web.
